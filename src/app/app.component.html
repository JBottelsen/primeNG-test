Accordion with Focus Out Directive

This tests the following functions

  • Clicking outside of a designated element triggers A function

  • Exclude a list of DOM items outside of element from triggering directive function

  • When a form is dirty, clicking outside of the accordion calls the directive function

  • Clicking outside when the form is pristine, nothing happens

  • If the form is dirty and the accordion is attempted to be closed, block user from closing the accordion and spawn modal
