Accordion with Focus Out Directive

This tests the following functions

  • Clicking outside of a designated element triggers A function

  • Exclude a list of DOM items outside of element from triggering directive function

  • When a form is dirty, clicking outside of the accordion calls the directive function

  • Clicking outside when the form is pristine, nothing happens

  • If the form is dirty and the accordion is attempted to be closed, block user from closing the accordion and spawn modal



Accordion with Focus Trap Directive

This demo demonstrates the following

  • When the accordion is closed, the directive is not applied so the user can still tab through the page as normal

  • When the accordion is open, the user is trapped inside the accordion when tabbing

  • When the form is dirty, clicking the "Save" button clears the form, disables the directive and closes the accordion which pops a modal informing the user their form data was accepted. The user can tab through the rest of the page as needed.

  • When the user clicks the cancel button, the accordion is closed and any remaining data in the form is cleared. The directive is also removed allowing the user to tab freely outside of the accordion.
